Data Donation Module Login

The Data Donation Module (DDM) is an application that researchers can use to collect data donations from citizens. DDM is hosted by the Data Donation Lab.

Members of the University of Zurich can use DDM by logging in with a Switch edu-ID account that is affiliated with the University of Zurich. Members of other institutions should get in touch with us.

If you are a UZH member, you can click on the button below and login/sign up. You will be redirected to the Switch edu-ID authentication page.

Once logged in, you can set up your data donation project and start your data donation collection. For more information on the application and how to use it, visit the following resources:

  • DDM Documentation
  • DDM Source Code
